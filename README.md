1. Yêu cầu cấu hình: php > 8.0 và node js > 21.0.0
2. cài đặt package:
  - ở back-end: composer install
  - ở front-end: npm install
3. cài đặt connect với database: cài đặt ở file back-end/.env
4. Chạy chương trình:
  - front-end: npm run serve
  - back-end: 
  + php artisan serve
  + php artisan storage:link
